Jeu de Mémoire

Comment Jouer

Bienvenue sur votre jeu de mémoire ! Suivez ces étapes pour jouer :

  1. Remplissez le formulaire avec votre nom, prénom et choisissez un niveau.
  2. Cliquez sur le bouton "Commencer" pour lancer le jeu.
  3. Retournez les cartes en cliquant dessus pour trouver les paires.
  4. Chaque niveau ajoute une paire de cartes.
  5. Réussissez tous les niveaux pour terminer le jeu.
  6. Et sutout, amusez-vous !!
